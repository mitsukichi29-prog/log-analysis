# Log Analysis Tool

## Overview
This tool analyzes log files and detects failed login attempts.

## Features
- Detect failed login
- Count IP addresses

## Usage
python3 log_analysis.py

## Sample Log

192.168.1.10 login success
192.168.1.20 login failed
192.168.1.20 login failed

## Sample Output

192.168.1.20 2
